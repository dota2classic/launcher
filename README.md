# D2C Launcher

Лаунчер для **Dota 2 Classic** — сообщественного сервера, запускающего оригинальную Dota 2 на движке Source 1.

---

## О проекте

Dota 2 Classic — это сообщественный проект по сохранению и поддержке старой версии Dota 2, которую Valve официально прекратила поддерживать. Лаунчер обеспечивает полный цикл для игры на сервере сообщества: аутентификацию через Steam, поиск матча и запуск игры.

**Бэкенд:** `https://api.dotaclassic.ru`

---

## Возможности

- **Аутентификация через Steam** — получение тикета аутентификации через вспомогательный процесс SteamBridge
- **Поиск матча** — очередь с обновлениями в реальном времени через Socket.IO
- **Управление пати** — отправка и принятие приглашений, выход из пати
- **Проверка и загрузка игры** — валидация и обновление файлов игры по манифесту
- **Запуск игры** — автоматический запуск с нужными аргументами Source 1
- **Настройки игры** — управление cvars (лимит FPS, консоль, автоатака и др.) с двусторонней синхронизацией `config.cfg`
- **Автообновление лаунчера** — обновления через Velopack с GitHub Releases без ручного скачивания

---

## Системные требования

- **ОС:** Windows 10/11 (x64)
- **Steam:** должен быть запущен для аутентификации
- **Игра:** папка с Dota 2 Classic (указывается при первом запуске)

---

## Быстрый старт

1. Скачайте последний релиз с [GitHub Releases](../../releases/latest)
2. Распакуйте и запустите `d2c-launcher.exe`
3. Убедитесь, что Steam запущен и вы вошли в аккаунт
4. Укажите папку с установленной Dota 2 Classic
5. Выберите режим игры и нажмите «Найти матч»

## Настройки игры

Лаунчер позволяет управлять основными игровыми параметрами прямо из интерфейса:

| Параметр | Cvar |
|----------|------|
| Лимит FPS | `fps_max` |
| Включить консоль | `con_enable` |
| Отключить зум камеры | `dota_camera_disable_zoom` |
| Принудительная атака правой кнопкой | `dota_force_right_click_attack` |
| Автоповтор правой кнопки | `dota_player_auto_repeat_right_mouse` |
| Сброс камеры при спавне | `dota_reset_camera_on_spawn` |
| Режим автоатаки | `dota_player_units_auto_attack` |
| Режим для дальтоников | (составной) |

Настройки синхронизируются двусторонне: изменения в игре подхватываются лаунчером, и наоборот.

## Сборка из исходников

### Требования

- .NET 10.0 SDK

### Сборка

```bash
dotnet build
```

### Запуск (отладка)

```bash
dotnet run --project d2c-launcher
```

### Публикация (production, win-x64)

```bash
dotnet publish d2c-launcher -c Release -r win-x64 --self-contained
dotnet publish SteamBridge -c Release -r win-x64 --self-contained -p:PublishTrimmed=true
```

### Тесты

```bash
dotnet test
```

## Стек технологий

| Слой | Технология |
|------|-----------|
| Язык | C# (.NET 10.0) |
| UI | Avalonia UI 11.x (Fluent, тёмная тема) |
| Паттерн UI | MVVM (CommunityToolkit.Mvvm) |
| Реальное время | SocketIOClient (Socket.IO / WebSocket) |
| Steam | Steamworks.NET + SteamBridge subprocess |
| API-клиент | NSwag (генерация из OpenAPI) |
| Обновления | Velopack |

---

## Структура проекта

```
d2c-launcher/        — основное приложение
  Views/             — Avalonia XAML-интерфейсы
  ViewModels/        — MVVM ViewModels
  Services/          — бизнес-логика (API, WebSocket, настройки, обновления)
  Integration/       — мониторинг Steam, консольные команды Dota
  Models/            — модели данных
  Generated/         — авто-сгенерированный API-клиент (не редактировать)
SteamBridge/         — вспомогательный процесс для Steam SDK
d2c-launcher.Tests/  — xUnit-тесты
```
